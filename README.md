# Iris Flower Classification

This project uses Machine Learning to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

## Algorithms Used
- Logistic Regression
- KNN
- Decision Tree
- Random Forest
- SVM
- Naive Bayes

## Best Accuracy
96.67% using Support Vector Machine (SVM)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Run Project

```bash
python iris_classification.py
